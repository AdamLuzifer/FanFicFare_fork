New Name and URL

This version changes the name of the project from FanFictionDownLoader(FFDL) to FanFicFare. Along with the name change, there are changes to the internal structuring of the project code, but not to the functionality.

If you had saved configuration (including your username/password for fic sites) in FFDL, you will need to copy it over to FanFicFare. Visit the FFDL User Config page, copy your settings and then paste them on the FanFicFare User Config page.

Changes:

  • Add chapter limits with URL by giving chapter range.
    Examples:
    • https://www.fanfiction.net/s/2565609/1/[4] Chapter 4 only
    • https://www.fanfiction.net/s/2565609/1/[6-10] Chapters 6, 7, 8, 9 & 10 only
    • https://www.fanfiction.net/s/2565609/1/[-10] Chapters 1-10 only
    • https://www.fanfiction.net/s/2565609/1/[150-] Chapters 150 and up only
    The chapter range will be included in the ebook title. Can be changed or disabled in config with title_chapter_range_pattern.
  • New base_xenforoforum adapter type. The details are complex enough that I've started a new wiki page for future reference. Please refer to it for more details.
  • New [base_efiction] and [base_xenforoforum] sections for common settings for eFiction Base and XenForoForum Base adapters respectively.
  • New site: forums.spacebattles.com base_xenforoforum adapter
  • New site: forums.sufficientvelocity.com base_xenforoforum adapter
  • New site: ninelives.dark-solace.org base_efiction adapter
  • New description_limit feature to explicitly limit the allowable length of the description.
  • Fixes for http://spikeluver.com
  • New Russian language site: masseffect2.in -- Thanks to PlushBeaver for adding this.

Questions? Check out our FAQs.

If you have any problems with this application, please report them in the FanFicFare Google Group. The previous version is also available for you to use if necessary.

A few additional things to know, which will make your life substantially easier:

  1. First thing to know: We do not use your Google login and password. In fact, all we know about it is your ID – password is being verified by Google and is absolutely, totally unknown to anyone but you.
  2. You can download fanfiction directly from your iPhone, Kindle or (possibly) other ebook reader.
  3. Downloaded stories are deleted after some time (which should give you enough of time to download it and will keep Google happy about the app not going over the storage limit).
  4. If you see some funny characters in downloaded Plain Text file, make sure you choose text file encoding UTF-8 and not something else.
  5. If you think that something that should work in fact doesn't, post a message to our Google Group. we also encourage you to join it so you will find out about latest updates and fixes as soon as possible
Otherwise, just have fun, and if you want to say thank you — use the contacts above.
